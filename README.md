# Voice Assistant Powered by OpenAI

This project is a voice-enabled AI assistant built using Python, Flask, and OpenAI APIs.  
It allows users to interact with the system using voice input, processes the speech into text, sends it to an AI model, and returns an intelligent response as both text and speech.

## Features

- Voice input using microphone
- Speech-to-text processing
- AI-powered responses using OpenAI
- Text-to-speech output
- Flask-based backend API
- Modular and easy-to-extend architecture

## Tech Stack

- Python
- Flask
- OpenAI API
- SpeechRecognition
- Text-to-Speech (TTS)
- HTML/CSS (for frontend if applicable)
